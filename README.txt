Block World 3D

Files included:
- index.html
- style.css
- game.js

How to use:
1. Unzip the folder.
2. Upload all files to the same folder on your website.
3. Open index.html in a modern browser.

Notes:
- Uses Three.js from a public CDN, so internet access is needed unless you later download Three.js locally.
- Works on desktop and mobile browsers.
- Desktop controls: WASD to move, mouse to look, left click remove, right click place, 1-4 select blocks.
- Mobile controls: left joystick move, right side drag to look, Place/Remove buttons and tap right side to interact.
